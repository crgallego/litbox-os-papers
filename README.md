# Litbox Papers

The Litbox Papers are the permanent intellectual archive for Litbox OS.

Litbox OS builds AI-native operating systems for founder-led companies.

This repository preserves the narrative, philosophy, frameworks, and field notes behind that mission.

## Core Line

> Your company cannot scale if the founder is the operating system.

## Editorial Rule

> Don't write the myth. Write the discovery.

## Structure

- `chronicles/` — narrative development of the philosophy
- `essays/` — polished public arguments
- `frameworks/` — reusable models
- `field-notes/` — real-time implementation observations
- `case-studies/` — implementation evidence
- `product-council/` — recurring decision logs
- `quotes/` — reusable lines and fragments
- `book/` — long-form source material
- `templates/` — content templates
