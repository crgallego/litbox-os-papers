# Master Codex Prompt — Litbox Papers

Create a new repository called `litbox-papers`.

This repo is the permanent intellectual archive for Litbox OS.

It is not a blog repo.

It is the living documentation system for the evolution of Litbox OS as a philosophy, methodology, product, and company.

## Mission

Preserve discovery before memory turns it into myth.

Core editorial rule:

> Don't write the myth. Write the discovery.

## Repository Purpose

This repo should contain:

- Chronicles
- Essays
- Frameworks
- Field Notes
- Case Studies
- Quotes
- Product Council records
- Book/source material
